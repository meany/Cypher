LEGEND

Body (of address) = The public key, or hashed shared data (like a coin). Either binary or textual form.
Wallet address = Typed address body. Binary form.
Network address = Typed address version and body. Binary form.
Tangram address = An address that users handle. Textual form.
Encode = Transforms a binary address into a Tangram address. The output has a textual form.
Decode = Transforms a Tangram address into a binary address. The output has a binary form.
TryVerify = Tries to verify an address but doesn't throw an exception if it fails.
VerifyThrow = Tries to verify an address and throws an exception if it fails.
